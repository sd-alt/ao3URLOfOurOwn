# ao3URLOfOurOwn
AO3网站 的一个提供反向代理配置的项目。大佬的https://github.com/ExcitedCodes/URLOfOurOwn的登录部分会显示会话超时。因此，增加域名+/setcookie页面，让用户可自行输入cookie实现登录。

搭建方式可以参照https://github.com/ExcitedCodes/URLOfOurOwn链接。
